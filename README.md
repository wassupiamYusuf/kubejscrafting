# kubejscrafting
makes minecraft a lot harder in crafting
to use, you need the following mods:
Create: Diesel Generators,
Create,
KubeJS Create,
KubeJS, and
No Tree Punching
how to install: put the kubejs folder you download from this repository in .minecraft with the required mods with forge in 1.20.1
